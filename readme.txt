Calculator.php  - класс калькулятора

Для основного калькулятора для хранения, передачи операндов использовался массив, и для того, чтобы контролировать типы 
элементов массива - операндов, был создан класс, при инициализации, которого проверяется целочисленные элементы или нет.

Для определения операторов использовался абстрактный класс - для определения общего поведения - проверка количества операндов и наличие метода вычисления операции, 
и производные классы, которые задавали непосредственно способы вычисления заданных операторов.
Не было уточнения по отрицательным числам для возведения в степень. 
Этот класс позволяет возводить в отрицательную степень, при этом результат уже не целый, а флоат.
Поэтому в классе оператора возведения в степень тип результата флоат, как общий случай числовых данных.

Скрипт демонстрации работы класса находится в корне проекта - CalcRun.php.
В нем же реализован логгер в виде анонимного класса.
Пример реализации простого логгера был взят из тутора http://www.tutorialspoint.com/php7/php7_anonymous_classes.htm, 
остальное - уже как придумала.
Мне логгер представился по аналогии с реализацией логгера в Питоне в виде декоратора, и получилось, что вызов калькулятора происходит как бы в обертке логгера. Поправьте, если не так. 
Файл лога, находится тоже в корне проекта log.txt - очень удобно смотреть что происходило. Оценила;))) Гораздо веселее, чем в консоле крутить.

Поскольку в основном калькуляторе не было возможности использовать по максимуму type hinting при передаче операндов - поскольку массив, 
сделала маленький классик AltCalculator.php с одним оператором AltMultiXtoY(находится в папке AltOperators), чтобы посмотреть как новый PHP
обрабатывает проверку типов при явном их указании. 
Демонстрация работы класса в файле AltCalcRun.php в корне проекта. Класс без логгера, просто дампит вызов метода вычисления, и приведены разные варианты вызова и обработка исключений.
